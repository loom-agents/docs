## Get Started

```bash
$ npm i loom-agents
```

```typescript
import { Agent } from 'loom-agents';

const agent = new Agent({
  name: "Loomie",
  purpose: "You're an AI agent made to automate tasks simply and fast!",
});

const result = await agent.run("Ready to get building?");
console.log(result.final_message);
```
